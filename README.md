# BroCachi 

## Aim of work 
This project is aiming to provide a simple profile about diffrent browser cahces in your machine, which can help you manage diffrent browser caches from one central space. The interactions with the tool will be as a CLI-based. 

## Protocol 
We use gRPC in the tool as main protocol to implement service-to-service communications where the tool act as chache data aggregator from diffrent services on the machine.
